## 🚀 CÓMO USAR

### Desde la UI de la Aplicación
1. Ve a cualquier retrospectiva completada
2. Haz clic en el botón **"TXT"** para exportación rápida
3. O usa **"Configurar"** para opciones avanzadas
4. El archivo se descargará con el nuevo diseño profesional automáticamente

### Opciones Disponibles
- **📊 Incluir estadísticas**: Resumen numérico elegante
- **👥 Incluir participantes**: Lista profesional de miembros
- **✍️ Incluir autores**: Metadata de creadores de tarjetas
- **🔗 Incluir grupos**: Visualización de agrupaciones
- **📋 Incluir notas del facilitador**: Observaciones importantes
- **🎯 Incluir elementos de acción**: Tareas y responsables
- **🧠 Incluir análisis de sentimientos**: IA insights (si disponible)
